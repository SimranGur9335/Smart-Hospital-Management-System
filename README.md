# 🏥 Smart Hospital Management System

A Java Swing and MySQL-based desktop application designed to streamline hospital operations and patient record management. The system provides an intuitive graphical user interface for managing patient information, database connectivity, and healthcare administration tasks efficiently.

## 🚀 Features

- User Authentication and Login System
- Patient Record Management
- Add New Patients
- Update Patient Details
- Delete Patient Records
- Search Patient Information
- MySQL Database Integration
- Java Swing Graphical User Interface
- Efficient Data Storage and Retrieval
- JDBC Connectivity

## 🛠️ Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- MySQL Connector/J

## 📂 Project Structure

- MainApp.java
- LoginGUI.java
- HospitalGUI.java
- DBConnection.java
- Patient.java
- PatientDAO.java
- TestDB.java
- mysql-connector-j-9.6.0.jar

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/SimranGur9335/Smart-Hospital-Management-System.git
cd Smart-Hospital-Management-System
```

### 2. Create Database

```sql
CREATE DATABASE hospitaldb;
```

### 3. Configure Database

Update MySQL credentials in `DBConnection.java`.

### 4. Compile Project

```bash
javac -cp ".;mysql-connector-j-9.6.0.jar" *.java
```

### 5. Run Application

```bash
java -cp ".;mysql-connector-j-9.6.0.jar" MainApp
```
